### What's safe in production

- Build is green, types pass, 114 tests pass.
- **DB-missing graceful degradation**: every list query wraps in `safeRead()` and renders a yellow banner instead of 500-ing if the DB is unreachable.
- **Brain surfaces env-gated**: missing `ANTHROPIC_API_KEY` falls back to deterministic templates; missing `OPENAI_API_KEY` returns 503 with a friendly message; agent loop reverts to slash-commands if `AGB_WA_AGENT` is off.
- **Webhook security**: Meta signature verified via constant-time HMAC. Per-sender rate limit (30/min, 200/day defaults). Daily token cap ($2.50/day at Sonnet defaults).
- **Silence rules wired**: `personal-only` tag excludes contact from Claude prompts; quiet hours suppress nudges; `AGB_BRAIN_DISABLED=1` is a global kill switch.
- **RLS migration ready**: `supabase/migrations/20260526120000_rls_owner_policies.sql` adds owner-scoped policies; one `psql -f` away.
- **Instrumentation**: every webhook/cron route wrapped in `withErrorCapture`; Sentry-compatible when `SENTRY_DSN` is set; structured `console.error` otherwise.
- **`pnpm verify`** walks all 9 env-gated surfaces and reports `active/paused/broken` with one-line diagnostics.

## What's NOT validated yet (honest list)

These are environment-specific things only real credentials can confirm. Not blockers for v1, but the 0.5 of the 10.0 score that's still hypothetical:

1. **Supabase schema parity with vanilla Postgres**. The schema applied cleanly to local Postgres 18; Supabase is also Postgres 15+, so this should be trivial, but the RLS policies + `auth.uid()` semantics need a real `psql -f` against your project to confirm.
2. **Real Anthropic responses on your data**. Tool calls work end-to-end against a stubbed upstream. Real re-intros + nudge briefings need an actual API key to confirm the draft quality.
3. **Webhook real traffic**. Postmark + WhatsApp adapters work against stubbed fetch. Real round-trip catches IP allowlist issues, DNS, WhatsApp's retry policy quirks.
4. **Cost at scale**. Estimated $0.015 per agent inbound message based on token counts. Actual cost depends on how chatty your usage is + whether prompt caching kicks in (it should, automatically, on Anthropic's API).
5. **Multi-tenant**. Cofounder hasn't onboarded; all queries are owner-scoped but the multi-user UX hasn't been exercised.
